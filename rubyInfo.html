Ruby

Learn about the dynamically-typed general purpose programming language that is Ruby.


Ruby was created in the mid 1990s by Yukihiro Matsumoto. It is "a careful balance between functional programming and imperative programming." It is most often used to create and support web applications, and has a lively community supporting it to this day.
Source: https://developer.oracle.com/learn/technical-articles/what-is-ruby
